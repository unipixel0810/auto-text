## 🔍 서비스 현황 분석

**Stitch**는 AI 자막 생성 + 멀티트랙 영상 편집 플랫폼이다.
현재 핵심 파이프라인(STT → AI 자막 → 편집 → 내보내기)은 완성되어 있으나,
자막의 **시각적 표현력**이 정적(static)에 머물러 있어 경쟁 서비스 대비 차별화 부족.

## 🎯 선정 기능: 자막 애니메이션 프리셋 시스템

### Why this feature?
1. **전환율 직결** — 무료 체험 시 "움직이는 자막"이 가장 강한 WOW 요소
2. **차별화** — 현재 경쟁 서비스와 가장 눈에 띄는 격차
3. **구현 범위 명확** — 프론트엔드 중심, DB/백엔드 의존도 낮음
4. **수익화 연계** — 프리미엄 프리셋을 페이월 뒤에 배치 가능

### 목표 지표
- 자막 편집 세션 중 애니메이션 적용률 **30% 이상**
- 프리미엄 전환율 **+15%** (애니메이션 프리셋 잠금 유도)
- 사용자 세션 시간 **+20%**

## 📦 기능 범위 (In Scope)

### MVP 애니메이션 프리셋 (8종)
| ID | 이름 | 설명 |
|----|------|------|
| `fade-in` | 페이드 인 | 투명 → 불투명 |
| `fade-out` | 페이드 아웃 | 불투명 → 투명 |
| `slide-up` | 슬라이드 업 | 아래→위 등장 |
| `slide-down` | 슬라이드 다운 | 위→아래 퇴장 |
| `pop` | 팝 | 작게→크게 튀어나오기 |
| `typewriter` | 타이핑 | 글자가 하나씩 타이핑 |
| `bounce` | 바운스 | 통통 튀는 등장 |
| `shake` | 쉐이크 | 강조 흔들기 |

### 구성 요소
- `SubtitleAnimationPanel` 컴포넌트 (우측 사이드바 탭 추가)
- `SubtitleItem.animation` 필드 추가
- Player에서 CSS animation 렌더링
- 타임라인 클립에 애니메이션 뱃지 표시

### Out of Scope (v2)
- 커스텀 키프레임 편집기
- 3D 변환 효과
- 음악 비트 싱크
