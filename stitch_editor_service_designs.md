## Design System

### Theme
- **Color Mode:** Dark (#1E1E1E background)
- **Accent Colors:** Light blue (#4da0ff, #3d93f5, #51a0fb variations)
- **Font:** Inter (sans-serif)
- **Roundness:** Rounded corners (8px)
- **Saturation:** 2

### Color Palette
- Primary Background: #1E1E1E
- Secondary Background: #252525
- Canvas Background: #000000
- Accent Blue: #4A9EFF / #4da0ff
- Text Primary: #FFFFFF
- Text Secondary: #B0B0B0
- Muted Text: #808080

### Typography
- Font Family: Inter
- Font Sizes: 11px - 24px range
- Font Weights: 400 (normal) to 700 (bold)
- Line Heights: 1.2 - 1.75

### Spacing
- Consistent padding: 8px, 12px, 16px, 24px, 32px, 48px
- Border Radius: 8px - 12px for buttons and panels
- Component spacing: Consistent vertical and horizontal gaps

### Components
- All components follow the dark theme
- Light blue accents for active states and primary actions
- Rounded corners throughout
- Clean, professional aesthetic
- Consistent iconography
